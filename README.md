# ⚡ SchedlyAI — AI Scheduling Assistant

> Your calendar on autopilot. SchedlyAI automatically schedules your meetings, organizes tasks, and protects your focus time.

## 🚀 Live Demo

### **👉 [https://schedly-ai.vercel.app](https://schedly-ai.vercel.app)**

---

## ✨ Features

- 🤖 **Smart Auto-Scheduling** — AI places meetings at optimal times
- 🛡️ **Focus Time Protection** — Blocks deep work sessions automatically
- 👥 **Team Coordination** — Multi-calendar, multi-timezone scheduling
- 💬 **Natural Language Input** — "Schedule a call with Sarah next Tuesday"
- ⚡ **Conflict Resolution** — Auto-reschedules based on priority rules
- 🔗 **Universal Sync** — Google Calendar, Outlook, Zoom, Meet & more

## 🛠️ Tech Stack

- **React** + **Vite**
- **Tailwind CSS v4**
- **Mobile-first responsive design**
- **Dark mode by default**

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
